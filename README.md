# Hospital_Emergency_Room_Dashboard

An interactive Excel dashboard for analyzing hospital emergency room data using charts, pivot tables, slicers, and KPIs.

## Tool Used
- Microsoft Excel
- Power Query
- Power Pivot (DAX)

## Features
- Patient Count
- Average Wait Time
- Patient Satisfaction Score
- Admission Status
- Gender-wise Analysis
- Age Group Analysis
- Department-wise Referrals
- Interactive Month & Year Filters
